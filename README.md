# 3DEditor
基于three.js+vite+element-plus开发的三维编辑器 

## 核心依赖版本

| 依赖名称| 版本号 | 
|---------|---------|
| node   | 18.20.5   | 
| npm   | 10.8.2    | 
| three.js   | r174    | 
| vue   | 3.5.13    | 
| vite   | 6.2.1    | 
| element-plus   | 2.3.1    | 
## 主要功能
1 小场景三维模型在线查看，包括更换天空盒

2 三维模型比例调整 

3 三维模型更换法线贴图、纹理贴图  

4 三维模型格式转换
